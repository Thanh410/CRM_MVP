Tóm tắt thay đổi so với bản hiện tại

Những điểm chính được redesign — không phải đổi mới hoàn toàn, mà nâng cấp.

🎨 Hệ màu
  • Aurora gradient (violet → cyan → pink) làm signature accent
  • Status pills bo tròn với màu pastel + ring nhẹ thay vì badge phẳng
  • Sidebar có aurora glow subtly chứ không đen tuyền
  • Stage colors có ngữ nghĩa: violet/indigo/cyan/amber/emerald
📐 Layout
  • Card radius lớn hơn (20px) cho cảm giác mềm mại
  • Bento grid cho dashboard (KPI 4 ô + funnel/source/activity/tasks)
  • Topbar có breadcrumb + quick action buttons
  • Toolbar cho list pages có view switcher (Bảng/Kanban/Card)
✨ Chi tiết
  • Plus Jakarta Sans cho headlines, Inter cho body
  • Avatar gradient (đa hue) thay vì 1 màu pastel
  • Sparkline có gradient stroke + animation reveal
  • Kanban column header có gradient theo stage
  • Inbox: chip kênh (FB/Zalo/Email) màu thương hiệu
  • RBAC matrix: cell có màu nền theo state (✓/○/—)
  • Greeting "Chào buổi sáng" có nhân hóa
Sau khi bạn duyệt mockup này
Mình sẽ viết file docs/UI-REDESIGN.md chứa: bộ design tokens mới (Tailwind config + CSS variables), spec từng component (props, variants, states), checklist migration theo từng page, và sample code cho các pattern khó (sidebar glass, sparkline gradient, kanban gradient header, RBAC matrix cell). File này dùng làm task list để build tuần tự.
